# Spec-Tech Engineering Prototype Lab Website

Static landing page for Spec-Tech Engineering Prototype Lab.

## Purpose

This website is designed as a simple landing page for capstone, thesis, prototype, and engineering project inquiries.

It highlights:

- Arduino / ESP32 / STM32 project support
- Raspberry Pi / Jetson Nano / AI vision projects
- Robotics and automation
- IoT dashboards and sensor systems
- Solar and LiFePO4 battery monitoring
- PCB, wiring, troubleshooting, and guided project mentoring

## Publishing with GitHub Pages

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Open **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch: **main**.
6. Select folder: **root**.
7. Save.

After GitHub Pages finishes deployment, the site can be used as a public link for the Spec-Tech Facebook page.
