Research in progress

SmartLink: STM32 + nRF24L01 telemetry link

SmartLink is an ongoing research project for stronger low-cost wireless telemetry using STM32F103C6 boards and nRF24L01+ PA/LNA modules. The goal is not only to send data, but to measure whether the link is healthy, degraded, unstable, or lost.

STM32F103C6 nRF24L01+ PA/LNA Arduino IDE Telemetry Link Health Drone / Robot Research

Current direction: build reusable Transceiver A and Transceiver B examples, then add optional health feedback for packet loss, duplicate detection, jitter, timeout, and failsafe behavior.

Conducted bench experiment:
  • Stable bench run: 5,000+ valid packets with 0 missed packets under controlled bench conditions.
  • Early diagnostic run: 3,600 valid packets, 13 duplicate packets, and 551 missed packets before tuning and RF handling improvements.
  • RF sensitivity check: holding or touching the nRF module body caused missed packets to rise up to 38, showing the importance of antenna clearance, enclosure design, grounding, and shielding.
  • Tracked variables: packet ID, valid count, missed count, duplicate count, packet interval, jitter, timeout, and health state.
